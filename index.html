Architecture du modèle — From Scratch

Pipeline d'entraînement
# 1. Split train/test (80/20, seed=42) x_train, x_test = 24 obs. / 6 obs. # 2. Normalisation z-score (sur train uniquement) x_n = (x − μ_x) / σ_x y_n = (y − μ_y) / σ_y # 3. Descente de gradient batch for iter in range(10_000): dw = Σ(ŷ − y)·x / m # gradient w db = Σ(ŷ − y) / m # gradient b w -= lr·dw (lr = 0.01) b -= lr·db if |Δcost| < 1e-9: break # 4. Dénormalisation → espace original w_orig = w_n · (σ_y / σ_x) b_orig = μ_y + σ_y·b_n − w_orig·μ_x # 5. Validation sklearn : Δw < 1e-4 ✓

⚠️
Taille du dataset
n = 30 : Attention
Avec seulement 30 observations, les métriques restent sensibles à la composition du split. Le k-fold corrige partiellement cette instabilité.
🌐
Extrapolation
Hors [1.2, 10.6] : Déconseillée
Le modèle est valide dans la plage d'entraînement. Toute prédiction hors bornes est extrapolation non garantie.
Synthèse des limites : Bien que le modèle soit statistiquement valide sur ses hypothèses, il faut noter que le salaire dépend de nombreuses variables omises (secteur d'activité, diplôme, localisation géographique, poste exact). Ce modèle est excellent comme introduction pédagogique à la régression linéaire from scratch — il ne doit pas être utilisé comme outil de décision RH.
